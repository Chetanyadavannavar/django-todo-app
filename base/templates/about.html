About TODO App

The TODO App is a simple and efficient task management application designed to help users organize their daily activities. It allows users to create, manage, and track their tasks easily.

With this application, users can add new tasks, mark tasks as completed, move tasks to trash, and manage their productivity in a structured way.

Features

  • Add and manage tasks
  • Mark tasks as completed
  • Move unwanted tasks to trash
  • User authentication (Login / Register)
  • Personal profile management

Purpose

The goal of this application is to help users stay organized, increase productivity, and keep track of important tasks in a simple and user-friendly way.

Technology Used

  • Django (Backend Framework)
  • HTML & CSS (Frontend)
  • SQLite / Database

This project demonstrates how a full-stack web application can be built using Django with user authentication and task management functionality.
